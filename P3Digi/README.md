Para poner en marcha este proyecto es necesario realizar tres configuraciones básicas: la base de datos, la API Key gratuita y la ejecución del programa. En primer lugar, la base de datos MySQL debe crearse utilizando el script incluido en base_de_datos.sql, que genera la base tareasdb y la tabla necesaria para almacenar las tareas. Una vez creada, es imprescindible comprobar que los datos de conexión del archivo ConexionBD.java coinciden con tu entorno local, utilizando el usuario root, la contraseña usbw y el puerto por defecto 3306. En segundo lugar, para habilitar la funcionalidad de inteligencia artificial, es necesario obtener una API Key gratuita desde la plataforma Groq: basta con registrarse en https://console.groq.com, acceder al apartado API Keys y generar una nueva clave. Esa clave debe copiarse en el archivo IAService.java, sustituyendo el valor de la variable API_KEY. Finalmente, para ejecutar el programa, solo hay que abrir el proyecto en IntelliJ IDEA, asegurarse de que la base de datos está creada y la API Key configurada, y ejecutar el archivo Main.java. Al iniciarse, el programa mostrará un menú en consola desde el que se pueden crear, listar, modificar, eliminar y buscar tareas, utilizando la IA para sugerir automáticamente la prioridad de cada una.
